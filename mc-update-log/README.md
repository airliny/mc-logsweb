# MC 服务器更新日志系统

一个美观的 Minecraft 服务器更新日志管理系统，采用毛玻璃（Glassmorphism）设计风格。

## ✨ 功能特性

- **前台展示** - 毛玻璃风格的更新日志展示页面，支持分类、版本筛选和搜索
- **后台管理** - 完整的后台管理系统，支持多用户协作
- **用户权限** - 管理员和编辑两种角色，权限分明
- **常用设置** - 快速修改站点名称、服务器 IP、官网链接、主题颜色等
- **官网链接按钮** - 支持设置服务器官网链接，前台显示醒目的「访问官网」按钮
- **密码修改** - 支持用户自行修改密码
- **主题定制** - 可视化主题编辑（颜色、透明度、模糊程度、深色/浅色模式）
- **背景图片** - 支持自定义背景图片和网站图标
- **SQLite 存储** - 无需数据库配置，即开即用
- **响应式设计** - 完美适配手机、平板和桌面端

## 🚀 快速开始

### 1. 环境要求
- PHP 7.4+
- SQLite3 扩展
- GD 扩展（用于图片上传）

### 2. 部署
将 `mc-update-log/` 目录放到你的 Web 服务器即可使用。

### 3. 访问
- **前台首页**：`http://你的域名/mc-update-log/`
- **后台管理**：`http://你的域名/mc-update-log/admin/`
- **默认管理员**：`admin / admin123`（首次使用请立即修改密码）

## 🎨 主题定制

在后台的 **常用设置** 页面可以快速修改：
- 网站标题和描述
- 服务器名称和 MC 版本
- 服务器 IP 和官网链接
- 主色调和背景色
- 毛玻璃模糊程度
- 深色/浅色模式切换

完整的主题设置（透明度、图片、高级颜色）在 **高级设置** 页面。

## 🔧 技术栈

- PHP + SQLite3（后端）
- CSS3（毛玻璃效果、Flexbox/Grid、CSS 变量）
- 响应式设计
